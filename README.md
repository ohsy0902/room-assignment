# 🏢 기숙사 방 배치 시스템

학교 기숙사(알바트로스, 베리타스, 명덕재) 학생 배치를 관리하는 웹 기반 시스템입니다.

## 🎯 주요 기능

- ✅ **3개 기숙사 지원** - 총 160개 호실, 640명 수용
- ✅ **CSV 파일 업로드** - 명렬표, 선호학생, 기피학생
- ✅ **자동 배치 시스템** - 전체/기숙사별/층별 범위 선택
- ✅ **드래그 앤 드롭** - 수동 배치 및 학생 교체
- ✅ **Excel 내보내기** - 배치 결과 다운로드

## 🚀 시작하기

1. **브라우저에서 열기**
   ```
   index.html 파일을 더블 클릭하거나 브라우저로 드래그
   ```

2. **명렬표 업로드**
   - "파일 업로드" 섹션에서 CSV 파일 선택
   - 샘플 파일: `sample_students.csv`

3. **선호/기파학생 업로드 (선택)**
   - 선호학생: `sample_preferred.csv`
   - 기피학생: `sample_avoided.csv`

4. **자동 배치 실행**
   - 배치 범위 선택 (전체/기숙사별/층별)
   - "🎲 자동 배치 실행" 버튼 클릭

5. **수동 조정**
   - 학생 카드를 드래그하여 다른 호실로 이동
   - 정원 및 기피학생 제약 자동 검증

6. **결과 내보내기**
   - "📊 Excel로 내보내기" 클릭
   - 기숙사별 시트가 포함된 Excel 파일 다운로드


## 📝 CSV 파일 형식

### 명렬표 (필수)
```csv
이름,학년,성별
김민준,1,남
이서준,1,남
박서연,2,여
```

### 선호학생 (선택)
```csv
자치위원학년,학생1,학생2,학생3,학생4
1,김민준,이서준,박도윤,최예준
2,박서연,김지민,이채원,정다은
```

### 기피학생 (선택)
```csv
학생이름,기피학생1,기피학생2
장선우,임도현,한준서
최유나,강서윤,조민서
```

## 🎨 기술 스택

## 💡 사용 팁

1. **선호학생 그룹**
   - 각 그룹은 4명으로 구성
   - 같은 방에 자동 배정
   - 기숙사/층/호실 단계별 지정 가능

2. **기피학생**
   - 학생당 최대 2명 지정 가능
   - 다른 방으로 자동 배정
   - 배정 불가시 경고 후 강제 배정

3. **드래그 앤 드롭**
   - 학생 카드를 클릭하여 드래그
   - 정원 초과 시 자동 방지
   - 기피학생 충돌 시 경고 표시

4. **통계 확인**
   - 헤더에서 실시간 통계 확인
   - 총 학생/배치 완료/미배치 표시

## 🔧 문제 해결

### 파일이 업로드되지 않을 때
- CSV 파일 형식 확인 (UTF-8 인코딩 권장)
- 컬럼명이 정확한지 확인
- 브라우저 콘솔에서 에러 메시지 확인

### 자동 배치가 안 될 때
- 명렬표가 업로드되었는지 확인
- 배치 범위가 선택되었는지 확인
- 호실 수용 가능 여부 확인

### 드래그가 안 될 때
- 학생 카드를 정확히 클릭했는지 확인
- 대상 호실에 빈 자리가 있는지 확인
